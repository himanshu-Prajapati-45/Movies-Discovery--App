# 🎬 MovieVerse — The Ultimate Cinematic Discovery App

**MovieVerse** is a premium, high-performance web application built with **React** and **Tailwind CSS** that allows users to discover trending movies, watch trailers, and manage their personal wishlist with a seamless, cinematic user experience.

## ✨ Features

### 💎 Premium UI/UX
- **Cinematic Hero Section**: Features high-definition backdrops with "Ken Burns" zoom animations and elegant text gradients.
- **Glassmorphic Navigation**: A modern, translucent navbar that blurs background content as you scroll.
- **Smooth Animations**: Powered by **Framer Motion** for fluid entrance effects, hover states, and page transitions.
- **Skeleton Loaders**: Professional loading states that eliminate layout shift and provide immediate visual feedback.

### 🎥 Cinematic Discovery
- **Embedded Trailers**: Watch official YouTube trailers directly inside the app using a sleek, custom-built modal — no redirects to YouTube.
- **Search & Filtering**: Real-time movie search with dedicated "No Results Found" handling and "Trending/Top Rated" categories.
- **Deep Info**: Detailed movie pages showing cast members, directors, genres, runtime, and production info.

### 💾 User Features
- **Dynamic Wishlist**: Save your favorite movies with a single click.
- **Data Persistence**: Your wishlist is automatically saved to **localStorage**, so it stays there even after you refresh or close the browser.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React (or SVG)
- **API**: TMDB (The Movie Database)
- **State Management**: React Context API
- **Routing**: React Router 7

## ⚙️ Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository
```bash
git clone https://github.com/himanshu-Prajapati-45/Movies-Discovery--App.git
cd Movies-Discovery--App
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up environment variables
Create a `.env` file in the root directory and add your TMDB API Key:
```env
VITE_TMDB_API_KEY=your_api_key_here
```

### 4. Run the development server
```bash
npm run dev
```
